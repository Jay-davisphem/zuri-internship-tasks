# Steps
```sh
python3 -m venv venv # creates a virtual environment

. venv/bin/activate # activates the created virtual environment 'venv'

pip install django # installs django framework

django-admin startproject davisphem # creates a project named 'davisphem'(my zuri username)
cd davisphem

python manage.py runserver # start the django builtin server

# Ctlr C to cancel the server
```
